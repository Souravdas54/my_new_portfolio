# 🚀 My Portfolio

A full-stack portfolio application built with modern web technologies. This project showcases my skills, projects, resume, and professional journey as a MERN Stack Developer.

---

## 📌 About The Project

This portfolio website is designed to present my technical skills, development projects, and professional information in a clean and responsive user interface.

The application follows a client-server architecture:

* **Frontend:** Built with Next.js, TypeScript, and Tailwind CSS.
* **Backend:** Built with Node.js, Express.js, TypeScript, and MongoDB.
* **Media Storage:** Cloudinary integration for image management.
* **Database:** MongoDB for storing application data.

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/Souravdas54/my-portfolio.git
```

### Frontend Setup

```bash
cd client
npm install
npm run dev
```

### Backend Setup

```bash
cd server
npm install
npm run dev
```
